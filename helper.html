How to install KabodUcraft

  1. Download and install Fabric Loader. Make sure to select the loader version displayed above.
  2. After downloading, press WIN+R then type "%appdata%".
  3. Navigate to \AppData\Roaming\.minecraft
  4. Delete all mods from the "mods" folder, and replace them with the mods provided in the zip file.
  5. Launch game and join server.
