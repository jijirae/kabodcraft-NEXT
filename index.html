How to Install Your Modpack

  1. Download and install Fabric Loader.
  2. Download the Your Modpack zip file from the link below.
  3. Extract the contents of the zip file to your Minecraft mods folder.
  4. Launch Minecraft with the Fabric profile.
  5. Enjoy your modded experience!
